Detailed Description

Klasa zapewniająca funkcje synchronizujące lokalną bazę danych z bazą LDAP.

Podczas synchronizacji dane w bazie powinny zostać uznane za niespójne, tzn. serwis powinien być chwilowo niedostępny z zewnątrz.

Synchronizuje bazę obiektów faculty.models.Employee z zewnętrzną bazą LDAP (modele models.LdapEmployee).

Uwaga - zrywane są zależności z klucza obcego faculty.models.Employee.organizationalUnit. Ustawiona flaga faculty.models.Employee.isLdapSynced oznacza jedynie, że zsynchronizowane są dane osobowe pracownika: imię, nazwisko, email, tytuł naukowy i stanowisko. Pozostałe pola są nullowane, aby przygotować je do następnych etapów synchronizacji (jak np. __syncOrganizationalUnits()).
